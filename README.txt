"A is for Amoeba" is a fun web-based text editor that wants to help you write creatively; whether it's code, fiction, or a dream diary of our collective unconscious!

////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

Save all the files to a directory then open "AmoebaEdit.html" with your favorite browser. If you'd like to see the built in audio features, run a local server (e.g. in the command line type "python -m SimpleHTTPServer 8000" (minus the quotes) then navigate to port 8000 in your broswer, URL = http://localhost:8000/.
